Player Setup Instructions

1: Add your players below. The "shortname" is used for the token on the map preview (instead of token image), use 2 character abbreviations (e.g. "TR"). For the "name field", it must match the name in Avrae for the commands to work.

2: Update the locations. If fog of war is turned on, it will auto-calculate what is visible based on the visibility range set in the map setup tab. Use the ➜ arrow to quickly copy move commands for Avrae.

3: For the command to add the token code to Avrae, select the player(s) and click [Setup Token] If a token was uploaded in the tokens tool in this site, it will automatically look up that token code.

NPC Setup Instructions

1: Add NPCs below. Similar to players, use short names for map display.

2: Use "Add NPC to Avrae" to generate combat initiative commands.

3: Setup tokens to apply images to the battlemap.

Monster Setup Instructions

1: Add monsters below. Use "Short" for the map identifier (e.g., GO1) and "Name" for the monster type (e.g., Goblin). The name field has autocomplete from your token database.

2: Use "Add Monster to Avrae" to generate !i madd commands.

3: Setup tokens to apply images to the battlemap.

Paste your map URL, or use the map search to select a map from the database
